# Prerequisite

nodeJS `v16.13.0`  
yarn `v1.22.17`  
npx `v8.1.0`

# Available Scripts

Run NEXT (Design Playground)

```
yarn dev
```

Run Storybook

```
yarn sbk
```

Build Storybook

```
yarn sbk-build
```

Build as distibution package

```
yarn pkg-build
```
